# Game Studio

## Repo Status

In winemaker04, this is not the default router.

Default router skill: `../winemaker-game/SKILL.md`

## Use This Skill Only When

- the user explicitly asks for engine or runtime routing (Phaser, Three.js, React Three Fiber, WebGL)
- the user asks to compare browser game engine options
- the task is specifically about external browser-game architecture outside the normal Winemaker app workflow

## Do Not Use This Skill For

- normal Winemaker feature work
- Winemaker UI and page work
- Winemaker services and database changes
- Winemaker debugging and verification

Use `../winemaker-game/SKILL.md` for those tasks.

## If Activated

1. Keep the request scoped to explicit engine-routing needs.
2. Route to specialist game skills only for the part that actually needs them.
3. Return control to `../winemaker-game/SKILL.md` once engine routing is resolved.
